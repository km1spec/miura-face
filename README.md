# 顔拡大

スマホ画面いっぱいに画像を表示するシンプルなHTMLです。

## ファイル構成

- `index.html`
- `image.jpg`

## 使い方

1. このフォルダに表示したい画像を `image.jpg` という名前で置きます。
2. `index.html` をスマホやブラウザで開きます。
3. 画像が画面全体に表示されます。

## GitHub へアップロード

```powershell
git init
git add .
git commit -m "Add fullscreen mobile image page"
git branch -M main
git remote add origin https://github.com/あなたのユーザー名/あなたのリポジトリ名.git
git push -u origin main
```

## 補足

- 画像名を変える場合は `index.html` の `src="./image.jpg"` を変更してください。
- GitHub Pages を使うとWeb公開もできます。
